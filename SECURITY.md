# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

**Do not open a public issue.**

Email: security@ezmode.games

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Time

We aim to respond within 48 hours and provide a fix within 7 days for critical issues.

## Scope

This policy applies to all ezmode-games repositories and services:
- ctd.ezmode.games
- api.ezmode.games
- All public repositories

## Recognition

We appreciate responsible disclosure. Contributors who report valid security issues will be credited in release notes (unless they prefer anonymity).
